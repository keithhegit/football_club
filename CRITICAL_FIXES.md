# 关键修复列表 - Match Engine Realism

## 🔴 问题1: 比分过高 (24-26)
**根本原因**: 射门转化率100%，每次射门=进球
**修复方案**: 
- 射门 base score过高 (Finishing 0.4权重 / 20 = ~0.7)
- 需要将射门基础概率降低到 0.15-0.25范围
- 增加守门员扑救系统

## 🔴 问题2: 传球成功率低 (27%)
**根本原因**: 防守压制过强 (1 - defenseScore = 0.3)
**修复方案**:
- 提高传球base score
- 降低防守压制效果 (max 0.8 -> max 0.4)

## 🔴问题3: Live模式崩溃
**根本原因**: LiveMatchPlayer使用错误 - 应该用新引擎但还在调用旧引擎代码
**问题**: "Team has no players" 说明旧引擎被调用了

## 🔴 问题4: 缺少犯规/黄红牌
**需要添加**: FOUL event type + card system

## 修复计划
1.降低射门成功率 (基础概率 x 0.3)
2. 提高传球成功率 (防御压制 / 2)
3. 添加犯规系统 (5-10次/场)
4. 修复 Live模式 (检查MatchEngine.simulateTick的返回值类型)
